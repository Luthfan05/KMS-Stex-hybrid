# 🔐 Kebijakan Keamanan Data

Panduan wajib bagi seluruh karyawan tentang perlindungan data dan keamanan sistem informasi perusahaan.

## Prinsip Dasar

Perusahaan menerapkan prinsip **"Need to Know"** — setiap karyawan hanya boleh mengakses data yang diperlukan untuk pekerjaannya.

## Aturan Password

| Ketentuan | Standar |
|---|---|
| Panjang minimal | 10 karakter |
| Karakter wajib | Huruf besar, huruf kecil, angka, simbol |
| Masa berlaku | Ganti setiap 90 hari |
| Riwayat | Tidak boleh mengulang 5 password terakhir |
| Multi-Factor Auth | Wajib untuk akun email & sistem inti |

:::danger Larangan Keras
- ❌ Berbagi password dengan siapapun, termasuk atasan
- ❌ Menyimpan password di sticky note/kertas
- ❌ Menggunakan password yang sama untuk akun pribadi dan kerja
:::

## Penggunaan Perangkat

### Laptop / Komputer Kantor
- Wajib **lock screen** saat meninggalkan meja (Win+L)
- Antivirus harus aktif dan up-to-date
- Tidak boleh menginstall software tanpa persetujuan IT

### Perangkat Pribadi (BYOD)
- Akses email kerja di HP wajib menggunakan **aplikasi resmi** (tidak via browser)
- Aktifkan **enkripsi perangkat** dan lock screen PIN
- Laporkan ke IT jika perangkat hilang/dicuri

## Insiden Keamanan

Laporkan segera ke IT jika:
- Menerima email mencurigakan (phishing)
- Perangkat diakses orang yang tidak berwenang
- Menemukan data sensitif di tempat tidak semestinya
- Password diduga bocor
